# AI Skills Management (AI 技能管理工具)

**AI-skills-management** 是一款基于 Electron + React + Vite 构建的现代化桌面应用程序。旨在为你提供一个优雅、高效且功能强大的可视化技能池总览管家，专门用于管理、下发以及云端同步你的 AI 本地技能配置（如 `.agents/skills` 等项目常用技能模板）。

## ✨ 核心特性

- 🎨 **极简 & 现代化的桌面 UI**
  - 支持浅色/深色主题无缝切换，伴有灵动酷炫的页面动效。
  - 清单数据清晰展示每一项技能包含的文件数、配置情况以及最后更新时间。
- 🗂️ **智能分类与标签系统**
  - 置顶固定主要类目分区（`全部`、`产品`、`设计`、`前端`、`后端`、`其他`），卡片上可自由搭配无穷多个作用备注（可视化阅读、开发必备...等）。系统会自动将技能聪明地归落到相应专栏下。
- 🌩 **一键私有云端同步 (Cloud Sync)**
  - 内置基于授权 Token 的后台跨设备同步体系。无论在什么设备上补充了新的能力库，只需一键点击**“推送到云端”**或**“从云端恢复”**，即刻完成私密无缝的代码云游。
- 🔗 **极速精准：特定子目录导入**
  - 支持直接粘贴第三方（如 Github）具体到某个深层技能文件的链接（如：`.../tree/main/skills/vue`）。应用底层利用了 Git Read-Tree 技术链优化提取机制，能瞬间将包裹极深的树状目录“摘选且扁平化平铺”在你的技能库根目录，彻底告别恶心的“套娃式”嵌套文件夹。
- ⚙️ **智能部署与防冲突检测**
  - 支持一键将指定的核心技能组下发配置到你的目标项目中；期间倘若遇到文件占用、覆写冲突时，将唤出直观的对话窗口交由你全盘接管。

## 🛠️ 技术栈

- **框架支持**： [Electron](https://www.electronjs.org/) + [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
- **UI 风格**：原生 CSS 结合部分样式组件，[Lucide-React](https://lucide.dev/) (图标体系)
- **底层管理**：内嵌集成了高度客制化的 Node.js 子进程调度及原生 Git 钩子，具备完整的权限、环境分离与智能树状文件提取能力。

## 🚀 极速上手

### 1. 环境准备

请确保你在本地安装了以下环境：
- Node.js (推荐 v18+)
- 预装 Git 并配置好基础运行环境

### 2. 下载与运行启动

```bash
# 1. 克隆代码库到本地
git clone https://github.com/chenxiaoyi0807/AI-skills-management.git

# 2. 进入项目目录
cd AI-skills-management

# 3. 安装全部依赖组件
npm install

# 4. 启动开发者调试模式
npm run dev

# 5. 一键构建用于生产环境的桌面客户端 (例如 Windows / macOS)
npm run build
```
